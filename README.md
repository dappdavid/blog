# DappDavid Blog

A minimal blog about the latest trends in tech.

## Features

*   Minimalist design with dark/light mode toggle.
*   Responsive layout.
*   RSS feed and Sitemap.
*   Archives by year.
*   Accessible markup.

## Local Development

1.  **Install Jekyll:** Follow the [official installation guide](https://jekyllrb.com/docs/installation/).
2.  **Clone the repository:**
    ```bash
    git clone https://github.com/dappdavid/blog.git
    cd blog
    ```
3.  **Run the server:**
    ```bash
    bundle install
    bundle exec jekyll serve
    ```
4.  **Visit the site:** Open `http://localhost:4000/blog/` in your browser.
